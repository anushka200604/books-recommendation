# Book Recommendation System

## Overview

This project implements a content-based book recommendation system using Natural Language Processing (NLP) techniques. The system recommends books similar to a selected book based on title, author, and description.

## Dataset

* 4766 books
* Features used:

  * Title
  * Authors
  * Description
  * Average Rating

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Methodology

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. TF-IDF Vectorization
5. Cosine Similarity
6. Recommendation Generation

## Results

The system successfully recommends books with similar themes, authors, and content based on textual similarity.
